## 시작하기

### 로그인
1. 웹사이트에 접속하면 로그인 화면이 표시됩니다.
2. 사용자명과 비밀번호를 입력합니다.
3. 로그인 버튼을 클릭합니다.

### 다크 모드
- 상단 메뉴바의 다크 모드 토글 버튼을 클릭하여 화면 테마를 변경할 수 있습니다.
- 설정은 자동으로 저장되어 다음 접속 시에도 유지됩니다.

## 주요 메뉴 소개

### 📊 대시보드
전체 통계 및 요약 정보를 한눈에 확인할 수 있는 메인 화면입니다.

**주요 기능:**
- 월별/일별 수익 추이 그래프
- 사이트별 통계
- 명의별 통계
- 전체 요약 정보

### 🌐 사이트 관리
사이트와 명의를 관리하는 핵심 메뉴입니다.

**주요 기능:**
- 명의 추가/수정/삭제
- 사이트 추가/수정/삭제
- 커뮤니티 관리
- 엑셀 일괄 등록
- 사이트명 통합 (관리자)

### 📋 사이트 정보 조회
사이트별 상세 정보를 조회할 수 있습니다.

### 💰 정산 관리
월별 정산 기록을 관리합니다.

**주요 기능:**
- 년월 선택
- 일별 정산 기록 입력/수정
- 명의별 정산 데이터 관리

### 🎲 메인 (DRBet)
충전/출금 기록을 관리하고 자동 출석을 처리합니다.

**주요 기능:**
- 일별 충전/출금 기록 입력
- 자동 출석 처리
- 출석일 자동 계산
- 명의별/사이트별 데이터 관리

### 🚀 시작
일일 작업 시작 시 사용하는 메뉴입니다.

### 🏁 마무리
일일 작업 마무리 시 사용하는 메뉴입니다.

### 💾 백업 관리 (슈퍼관리자 전용)
데이터베이스 백업을 관리합니다.

### 🏢 사무실 관리 (관리자 전용)
사무실 정보 및 텔레그램 설정을 관리합니다.

## 기능별 상세 가이드

### 1. 사이트 관리

#### 명의 관리

**명의 추가:**
1. 상단의 "➕ 새 명의 추가" 버튼을 클릭합니다.
2. 다음 정보를 입력합니다:
   - 성함 (필수)
   - 생년월일 (필수)
   - 은행 계좌 (여러 개 추가 가능)
   - 전화번호 (여러 개 추가 가능)
   - 닉네임 (여러 개 추가 가능)
   - 메모
3. "저장" 버튼을 클릭합니다.

**명의 수정:**
1. 명의 목록에서 수정할 명의를 클릭합니다.
2. 모달 창에서 정보를 수정합니다.
3. "저장" 버튼을 클릭합니다.

**명의 삭제:**
1. 명의 목록에서 삭제할 명의 옆의 "🗑️" 버튼을 클릭합니다.
2. 확인 메시지에서 "확인"을 클릭합니다.

**명의 순서 변경:**
- 명의 목록에서 드래그 앤 드롭으로 순서를 변경할 수 있습니다.

#### 사이트 관리

**사이트 추가:**
1. 명의를 선택합니다.
2. "🌐 새 사이트 추가" 버튼을 클릭합니다.
3. 사이트 정보를 입력합니다:
   - 사이트명 (필수)
   - 도메인
   - 경로-코드
   - 아이디
   - 비번
   - 환비
   - 닉네임
   - 경로
   - 장
4. "저장" 버튼을 클릭합니다.

**사이트 수정:**
- **인라인 편집 (빠른 수정):**
  - 테이블에서 수정할 셀을 클릭하거나 더블클릭합니다.
  - 값을 입력한 후 Enter 키를 누르거나 다른 셀을 클릭하면 자동 저장됩니다.
  
- **상세 편집:**
  - 사이트 행의 "✏️" 버튼을 클릭합니다.
  - 모달 창에서 모든 정보를 수정할 수 있습니다.

**사이트 삭제:**
1. 사이트 행의 "🗑️" 버튼을 클릭합니다.
2. 확인 메시지에서 "확인"을 클릭합니다.

**사이트 순서 변경:**
- 같은 승인유무 상태 내에서 드래그 앤 드롭으로 순서를 변경할 수 있습니다.

**승인유무 수정:**
- **더블클릭:** 일반 편집 모드
- **우클릭:** 수동 입력 모드
- 날짜는 MM.DD 형식으로 입력합니다 (예: 12.25)
- 편집 모드 진입 시 기본값은 오늘 날짜입니다.

**사이트 정보 복사:**
- 사이트 행의 "📋" 버튼을 클릭하면 사이트 정보가 클립보드에 복사됩니다.

#### 커뮤니티 관리

**커뮤니티 추가:**
1. "🌐 새 커뮤니티 추가" 버튼을 클릭합니다.
2. 커뮤니티 정보를 입력합니다.
3. "저장" 버튼을 클릭합니다.

**커뮤니티 수정/삭제:**
- 사이트와 동일한 방식으로 관리할 수 있습니다.

#### 엑셀 일괄 등록

1. "📋 엑셀 일괄 등록" 버튼을 클릭합니다.
2. 엑셀 형식에 맞춰 데이터를 입력합니다.
3. "일괄 등록" 버튼을 클릭합니다.

**엑셀 형식:**
- 각 행은 하나의 사이트/커뮤니티를 나타냅니다.
- 열 순서: 사이트명, 도메인, 경로-코드, 아이디, 비번, 환비, 닉네임, 승인유무, 경로, 장

#### 사이트명 통합 (관리자 전용)

1. "🔗 사이트명 통합" 버튼을 클릭합니다.
2. 통합할 사이트명 그룹을 확인합니다.
3. 원본 사이트명과 대상 사이트명을 선택합니다.
4. "통합 실행" 버튼을 클릭합니다.

**주의사항:**
- 사무실 관리자는 자신의 사무실 내 데이터만 통합할 수 있습니다.
- 통합된 사이트명은 되돌릴 수 없으므로 신중하게 진행하세요.

#### 사이트 정보 편집 (우클릭 메뉴)

- 사이트명 셀에서 우클릭하면 사이트 정보 편집 모달이 열립니다.
- 이벤트, 요율, 출석 설정 등을 관리할 수 있습니다.

---

### 2. 정산 관리

#### 정산 기록 입력

1. 상단에서 년월을 선택합니다.
2. 해당 날짜의 행을 더블클릭합니다.
3. 다음 정보를 입력합니다:
   - 수익 (자동 계산)
   - 승인 여부
   - 사이트/내용
   - 명의별 데이터 (충전, 출금, 출석일 등)
4. Enter 키를 누르거나 다른 셀을 클릭하면 자동 저장됩니다.

**날짜 입력:**
- 날짜 필드는 일자만 입력하면 됩니다 (년월은 이미 선택되어 있음).
- 예: 1일이면 "1"만 입력

#### 정산 기록 수정

- 수정할 셀을 더블클릭합니다.
- 값을 수정한 후 Enter 키를 누르거나 다른 셀을 클릭합니다.

---

### 3. 메인 (DRBet)

#### 충전/출금 기록 입력

1. 상단에서 날짜를 선택합니다.
2. "➕ 행 추가" 버튼을 클릭합니다.
3. 다음 정보를 입력합니다:
   - 명의 (4개까지)
   - 사이트명 (각 명의별)
   - 충전/출금 금액
4. Enter 키를 누르면 자동 저장됩니다.

**자동 출석:**
- 출석구분이 "자동"인 사이트의 경우, 충전금액이 변경되면 자동으로 출석일이 증가합니다.
- 출석구분은 사이트 정보 편집에서 설정할 수 있습니다.

#### 기록 수정

- 수정할 셀을 더블클릭합니다.
- 값을 수정한 후 Enter 키를 누르거나 다른 셀을 클릭합니다.

---

### 4. 사무실 관리 (관리자 전용)

#### 사무실 추가/수정

1. "➕ 새 사무실 추가" 버튼을 클릭합니다.
2. 사무실 정보를 입력합니다:
   - 사무실명
   - 관리자 계정
   - 상태
   - 설명
   - 주소
   - 전화번호
   - 메모
3. "저장" 버튼을 클릭합니다.

#### 텔레그램 설정

1. 사무실 목록에서 텔레그램 아이콘을 클릭합니다.
2. 텔레그램 봇 토큰과 채팅 ID를 입력합니다.
3. "저장" 버튼을 클릭합니다.

**텔레그램 봇 토큰 얻는 방법:**
1. 텔레그램에서 @BotFather를 검색합니다.
2. /newbot 명령어로 새 봇을 생성합니다.
3. 봇 토큰을 복사합니다.

**채팅 ID 얻는 방법:**
1. 텔레그램에서 @userinfobot을 검색합니다.
2. 봇과 대화를 시작하면 채팅 ID를 받을 수 있습니다.
3. 그룹 채팅의 경우, 봇을 그룹에 추가한 후 채팅 ID를 확인합니다.

## 키보드 단축키

### 사이트 관리 / 커뮤니티 관리

**셀 편집:**
- **Enter:** 현재 셀 저장
- **Escape:** 편집 취소
- **Tab:** 현재 셀 저장 후 다음 필드로 이동
- **Shift + Tab:** 현재 셀 저장 후 이전 필드로 이동

**필드 순서 (사이트):**
1. 도메인
2. 경로-코드
3. 아이디
4. 비번
5. 환비
6. 닉네임
7. 경로
8. 장

**필드 순서 (커뮤니티):**
1. 도메인
2. 경로-코드
3. 아이디 (name)
4. 비번 (user_id)
5. 환비 (password)
6. 환비 (exchange_password)
7. 닉네임
8. 경로 (path)

**자동 저장:**
- 다른 필드를 클릭하거나 편집 모드를 활성화하면 현재 편집 중인 필드가 자동으로 저장됩니다.

## 자주 묻는 질문

### Q1. 사이트를 추가했는데 목록에 나타나지 않아요.
**A:** 명의를 선택했는지 확인하세요. 명의를 선택하지 않으면 사이트가 표시되지 않습니다.

### Q2. 출석일이 자동으로 증가하지 않아요.
**A:** 사이트 정보 편집에서 출석구분이 "자동"으로 설정되어 있는지 확인하세요. "수동"으로 설정되어 있으면 자동으로 증가하지 않습니다.

### Q3. 사이트명 통합이 안 돼요.
**A:** 사무실 관리자는 자신의 사무실 내 데이터만 통합할 수 있습니다. 슈퍼관리자는 모든 데이터를 통합할 수 있습니다.

### Q4. Tab 키로 필드 이동이 안 돼요.
**A:** 편집 모드가 활성화된 상태에서 Tab 키를 눌러야 합니다. 편집 모드가 아닌 경우 셀을 먼저 클릭하세요.

### Q5. 승인유무 날짜를 수정하고 싶어요.
**A:** 승인유무 셀을 더블클릭하거나 우클릭하여 편집 모드로 진입한 후, 날짜 입력 필드에서 MM.DD 형식으로 수정하세요.

### Q6. 명의의 닉네임을 여러 개 추가하고 싶어요.
**A:** 명의 수정 모달에서 "➕ 닉네임 추가" 버튼을 클릭하여 여러 개의 닉네임을 추가할 수 있습니다.

### Q7. 정산 관리에서 날짜를 입력할 때 년월을 입력해야 하나요?
**A:** 아니요. 년월은 상단에서 이미 선택되어 있으므로, 날짜 필드에는 일자만 입력하면 됩니다 (예: 1, 15, 31).

### Q8. 다크 모드 설정이 저장되지 않아요.
**A:** 브라우저의 쿠키/로컬 스토리지 설정을 확인하세요. 쿠키가 차단되어 있으면 설정이 저장되지 않을 수 있습니다.

## 팁과 요령

### 효율적인 데이터 입력
1. **Tab 키 활용:** 필드 간 빠른 이동을 위해 Tab 키를 적극 활용하세요.
2. **인라인 편집:** 테이블에서 직접 수정하면 모달을 열지 않고도 빠르게 수정할 수 있습니다.
3. **일괄 등록:** 여러 사이트를 한 번에 추가할 때는 엑셀 일괄 등록 기능을 사용하세요.

### 데이터 관리
1. **정기적인 백업:** 슈퍼관리자는 정기적으로 백업을 수행하세요.
2. **사이트명 통합:** 유사한 사이트명은 통합하여 데이터 일관성을 유지하세요.
3. **명의 관리:** 명의 정보(은행 계좌, 전화번호, 닉네임)를 정기적으로 업데이트하세요.

### 성능 최적화
1. **필터링 활용:** 명의를 선택하여 필요한 데이터만 표시하세요.
2. **페이지네이션:** 많은 데이터가 있을 때는 페이지네이션을 활용하세요.

## 문제 해결

### 로그인이 안 돼요
- 사용자명과 비밀번호를 확인하세요.
- 계정이 비활성화되어 있지 않은지 확인하세요.
- 관리자에게 문의하세요.

### 데이터가 저장되지 않아요
- 인터넷 연결을 확인하세요.
- 브라우저를 새로고침하세요.
- 에러 메시지를 확인하고 관리자에게 문의하세요.

### 화면이 깨져 보여요
- 브라우저를 최신 버전으로 업데이트하세요.
- 브라우저 캐시를 삭제하세요.
- 다른 브라우저에서 시도해보세요.
